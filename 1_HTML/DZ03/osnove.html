Povratak

1. Fleksibilni model kutija (Flexbox)

Flexbox je jednodimenzionalni model rasporeda dizajnirn za distribuciju prostora između stavki u spremniku i moćnu mogućnost poravnanja. Idealan je za navigaciju, male komponente i centriranje elemenata.

2. Dvodimenzionalni model mreže (CSS Grid)

CSS Grid je dvodimenzionalni sustav rasporeda (redovi i stupci). Savršen je za dizajniranje cjelokupnog rasporeda stranice (main layout) gdje želite pozicioniranje elemenata.

3. Klasični rasporedi (Float & Columns)

Prije Flexboxa i Grida, koristio se Float za bočno pozicioniranje elemenata. **Multi-column** omogućuje pakiranje sadržaja u stupce.

Float lijevo

Ovaj tekst okružuje float element s lijeve strane. Klasično pozicioniranje pomoću float-a pomiče elemente ulijevo ili udesno dok dopušta da se okolni tekst omota oko njega. Također, ovaj odlomak je podijeljen u tri stupca pomoću svojstva **columns**, što je korisno za novinske ili tiskane rasporede.
